# Abschalt-Timer
Ein Timer um den Windows-PC nach einer angegebenen anzahl von Minuten abzuschalten!

Die Windows-Eingabeauforderung (CMD) dient als GUI.

Sollte auf folgenden Windows-Versionen laufen:
Windows 8
Windows 8.1
Windows 10
